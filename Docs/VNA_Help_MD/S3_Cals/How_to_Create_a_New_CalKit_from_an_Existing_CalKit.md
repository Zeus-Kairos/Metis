# How to Create a New Cal Kit from an Existing Cal Kit

* * *

You can create a new custom Cal Kit using an existing Cal Kit as a starting
point.

Here is how:

  1. Press Cal > Cal Sets & Cal Kits > Cal Kit....

  2. Immediately click Save... and change the file name. Select either *.xkt or *.ckt, *.prn file type. [Learn more about these file types.](VNA_Cal_Kit_File_Types.md)

  3. Make modifications to your new custom Cal Kit as required.

  4. Routinely save your work by clicking Save.

## To get your 8505xG calkit file

####

#### Option CDA, Characterized Databased Cal Kit.

The charcterized databased cal kit file offers the calibation data for your
own cal kit to provide much accurate calibration factor. Each cal kit is
calibated at factory shipment and a dedicated calibration data is stored.

  1. Visit [Keysight Service and warranty status](https://support.keysight.com/KeysightdCX/s/service-and-warranty-status) website. Enter the model numbe and serial number of your cal kit.

  2. Click "Chracteize Data" from the shown list to download the degingnated .xkt file. 

#### Nominal Databased Cal Kits

The 8505xG is not listed in Cal Kit selection at old firmware. In that case,
the .xkt file is available at the product suppot page.

  1. Visit the 8505xG product support website, ex: <http://www.keysight.com/support/85052g>. Select "Driver, Firmware and Software" tab. Find xkt file and download it.
