# Save and Load

The L8990M states can be divided into 2 categories: app settings and per-
channel settings. The L8990M configuration and characterization are app
settings. Switch path is per-channel setting.

  * "s2p" files

  * For each possible vna-port-to-test-port path, there is a s2p file (for example, L8990M-0LZ 20-port configuration, there are 32 different s2p files).

  * .swp file (content in xml format)

  * Switch matrix properties

  * Link to the s2p file paths

  * VNA state file

  * L8990M external device configuration

  * VNA state can contain multi channels, each channel contains its own:

  * Switch path selections

  * Test Port Labels

  * De-embedding settings
