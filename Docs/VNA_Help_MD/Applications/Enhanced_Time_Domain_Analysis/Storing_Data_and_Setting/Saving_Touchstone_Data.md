## File Extension (SnP Format)

The [SnP file format](../../../S5_Output/SaveRecall.md#sNp) is changed
depending on the selected DUT topology.

File Extension |  DUT Topology  
---|---  
.s1p |  Single Ended 1  
.s2p |  Single Ended 2, Differential 1  
.s4p |  Single Ended 4, Differential 2  
  
See the [Saving data in Touchstone
format](../../../S5_Output/SaveRecall.htm#sNp) for touch stone format.

## Saving Touchstone Data

The trace data can be saved in touch stone format.

  1. Click File in the tool bar.

  2. Select Save Touchstone from menu, then the Save Touchstone As dialog is displayed.

  3. Type desired file name.

  4. Click Save.
