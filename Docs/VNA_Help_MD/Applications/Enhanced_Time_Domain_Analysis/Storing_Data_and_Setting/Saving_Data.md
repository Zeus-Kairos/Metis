## Saving in .csv format

The trace data can be saved in .csv format.

  1. [Select the trace](../Making_Measurements/Setting_Up_Parameters_on_Each_Trace.md#Selecting_Trace) which you want to get the data.

  2. Click File in the tool bar.

  3. Select Save Active Trace Data or Save All Trace Data from menu, then the Save Trace As dialog is displayed.

  4. Type desired file name.

  5. Click Save.

Example of File

# Channel 1 # Trace 4 Frequency Formatted Data Formatted Data 8.49E+06
-5.84E-02 0.00E+00 1.70E+07 -8.27E-02 0.00E+00 2.55E+07 -1.04E-01 0.00E+00
3.40E+07 -1.32E-01 0.00E+00 4.25E+07 -1.52E-01 0.00E+00 5.09E+07 -1.84E-01
0.00E+00 5.94E+07 -2.18E-01 0.00E+00 6.79E+07 -2.70E-01 0.00E+00 . (data is
continued until the end) .

## Saving in the other format

The trace data can be saved in .csv format.

  1. Click File in the tool bar.

  2. Select Save Data As, then the Save Trace As dialog is displayed.

  3. Select Save As Type.

  4. Type desired file name.

  5. Click Save.

See [Save Other Format](../../../S5_Output/SaveRecall.md#Save_Other) Topic
for more detail
