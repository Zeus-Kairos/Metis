# Available Masks

The following tables list the available standard masks files that you can use.
The TDR uses the same format as the Infiniium DCA (86100C). The following
available masks are the same as ones stored in DCA and stored under C:\Program
Files (x86)\Keysight\Network Analyzer\masks.

  * SDH/SONET

  * Fibre Channel

  * Ethernet

  * Infiniband

  * Serial ATA

  * XAUI

  * Electrical

  * G_984_2

[Other topics about Eye Diagram and Mask Test](Eye_Diagram_and_Mask_Test.md)
