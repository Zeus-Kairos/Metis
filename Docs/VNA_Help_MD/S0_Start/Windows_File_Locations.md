# Windows File Locations

* * *

With Windows 10, data file locations have been changed from previous Windows
Operating Systems.

### State files and most data file storage locations:

  * D:\, C:\

### Firmware executable file locations:

  * c:\program files\Keysight\network analyzer

### Support file locations

  * c:\programdata\Keysight\network analyzer

These file locations can be queried remotely.

SCPI Command: [SYSTem:CONFigure:DIRectory?](../Programming/GP-
IB_Command_Finder/System.htm#ConfDir)

COM Command: [DirectoryPath
Property](../Programming/COM_Reference/Properties/DirectoryPath_Property.htm)
