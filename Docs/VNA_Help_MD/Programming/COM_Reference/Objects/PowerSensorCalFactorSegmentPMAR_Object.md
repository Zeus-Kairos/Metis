# PowerSensorCalFactorSegmentPMAR Object

* * *

Description

Contains the properties describing a segment of a power sensor cal factor
table that is used with Power Meter as Receiver.

### Accessing the PowerSensorCalFactorSegment PMAR object

You can get a handle to one of these segments through
[CalFactorSegmentsPMAR](CalFactorSegmentsPMAR_Collection.md).Item(n).
