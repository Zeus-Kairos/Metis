# PowerLossSegmentPMAR Object

* * *

### Description

Contains the properties describing a segment of the power loss table used with
a Power Meter as Receiver.

### Accessing the PowerLossSegmentPMAR object

You can get a handle to one of these segments through
[PowerLossSegmentsPMAR](PowerLossSegmentsPMAR_Collection.md).Item(n)
