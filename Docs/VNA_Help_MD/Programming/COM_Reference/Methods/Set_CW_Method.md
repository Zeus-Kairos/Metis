## SetCW Method

* * *

#### Description

|  Changes the analyzer to sweep type CW mode and sets the CW frequency to the
marker's frequency. Does not change anything if current sweep type is other
than a frequency sweep.  
---|---  
  
####  VB Syntax

|  mark.SetCW
