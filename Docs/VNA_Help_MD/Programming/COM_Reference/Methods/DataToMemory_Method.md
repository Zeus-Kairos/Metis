## DataToMemory Method

* * *

#### Description

|  Stores the active measurement data into memory creating a memory trace. The
memory can then be displayed or used in calculations with the measurement
data.  
---|---  
  
####  VB Syntax

|  meas.DataToMemory  
  
#### Variable

|
