## LoadTheme Method

* * *

#### Description

| Load a color theme from a disc file.  
---|---  
  
#### VB Syntax

| colors.LoadTheme (filename)  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
colors | A [ComColors](../Objects/ComColors_Object.md) (object)  
filename | (String) \- Path and filename of the theme to load.  
  
#### Return Type

| Not Applicable  
  
#### Default

| Not Applicable  
  
#### Examples

| colors.LoadTheme = ("c:\Program Files(x86)\Keysight\Network
Analyzer\Colors\Theme1.colors")  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

| HRESULT LoadTheme(BSTR filename);  
  
#### Interface

| IColors
