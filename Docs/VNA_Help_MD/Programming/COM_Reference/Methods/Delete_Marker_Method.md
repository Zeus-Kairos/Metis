## DeleteMarker Method

* * *

#### Description

|  Deletes a marker from the measurement.  
---|---  
  
####  VB Syntax

|  meas.DeleteMarker(Mnum)  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
meas |  A Measurement (object)  
Mnum |  (long) - Any existing marker number in the measurement  
  
#### Return Type

|  Not Applicable  
  
#### Default

|  Not Applicable  
  
#### Examples

|  meas.DeleteMarker(1)  
  
#### [C++](../../Learning_about_COM/c++_and_the_com_interface.md) Syntax

|  HRESULT DeleteMarker(long lMarkerNumber)  
  
#### Interface

|  IMeasurement
