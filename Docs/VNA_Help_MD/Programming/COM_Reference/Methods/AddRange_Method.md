# AddRange Method

* * *

#### Description

|  Adds a frequency range using the next available range name. For example,
with only the default F1 range name present, sending this command will create
F2. On the [Measurement Setup
dialog](../../../Applications/Differential_IQ.htm#SetupDiag) this is the New
setting.  
---|---
