## Hold (channels) Method

* * *

#### Description

|  Places ALL channels in hold mode. To resume all channels sweeping, use
[chans.Resume](Resume_Method.md). (Must be the same instance of chans). To
place a single channel in hold mode, use [channel.Hold](Hold_Method.md)
Method.  
---|---  
  
#### VB Syntax

|  chans.Hold  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description
