## RemoveChannelNumber Method

* * *

#### Description

|  Deletes a channel by specifying the channel number. Use [Remove
Method](Remove_Method.htm) to delete a channel by specifying the index in the
channels collection.  
---|---  
  
####  VB Syntax

|  chans.RemoveChannelNumber (chan}  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
chans |  A [Channels](../Objects/Channels_Collection.md) (collection)  
(chan) |  The channel number to delete.  
  
#### Return Type

|  Not Applicable  
  
#### Default

|  Not Applicable  
  
#### Examples

|  chan.RemoveChannelNumber(2)  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT RemoveChannelNumber(VARIANT channelNumber)
