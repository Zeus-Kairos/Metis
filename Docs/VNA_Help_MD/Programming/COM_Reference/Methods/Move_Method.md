## Move Method

* * *

#### Description

|  Moves a trace from one window to another.  
---|---  
  
####  VB Syntax

|  trace.Move (winTo)  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
trace |  [Trace](../Objects/Trace_Object.md) (object)  
winTo |  (Long integer) Window number to move the trace to. If the window number does not exist, it will be created.  
  
#### Return Type

|  Not Applicable  
  
#### Default

|  Not Applicable  
  
#### Examples

|  trace.Move 2 'Moves the trace to window 2  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT Move()  
  
#### Interface

|  ITrace3
