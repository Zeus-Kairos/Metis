# DeleteParameter Method

* * *

#### Description

|  Deletes the named parameter.  
---|---  
  
####  VB Syntax

#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description

name |  (String) Parameter name that was used when the parameter was created.
