## Reset Method (PhaseRef)

* * *

#### Description

|  Resets all properties associated with the Phase Reference Cal to their
default values. For predictable results, send this command before performing
each Phase Reference Cal.  
---|---  
  
####  VB Syntax

|  phaseRef.Reset
