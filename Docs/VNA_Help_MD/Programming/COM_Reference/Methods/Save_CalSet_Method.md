## Save Method

* * *

#### Description

|  Saves the current Cal Set to disk. This is the recommended method for
saving a Cal Set. Learn more about [reading and writing Cal data using
COM](../../Learning_about_COM/Read_and_Write_Calibration_Data_using_COM.htm)  
---|---
