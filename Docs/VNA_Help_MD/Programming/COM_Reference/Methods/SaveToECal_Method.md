## SaveToECal Method

* * *

#### Description

|  Saves the User Characterization to the ECal module. This can take several
minutes to complete. To save to VNA disk memory, use [SaveToDiskMemory
Method](SaveToDiskMemory_Method.htm). User Characterization can be saved to
both VNA disk memory and ECal module memory.  
---|---
