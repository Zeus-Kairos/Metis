## ShowTitleBars Method

* * *

#### Description

|  Shows and Hides the Title Bars. The Title Bars are across the top of the
Network Analyzer Window and each of the measurement windows. The Window name
is shown in the Title Bar.  
---|---  
  
####  VB Syntax

|  app.ShowTitleBars state  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
app |  An [Application](../Objects/Application_Object.md) (object)  
state |  (boolean)  
True (1) \- Show the Title Bars  
False (0) \- Hide the Title Bars  
  
#### Return Type

|  Not Applicable  
  
#### Default

|  Not Applicable  
  
#### Examples

|  app.ShowTitleBars True  
  
#### [C++](../../Learning_about_COM/c++_and_the_com_interface.md) Syntax

|  HRESULT ShowTitleBars/(VARIANT_BOOL bState)  
  
#### Interface

|  IApplication
