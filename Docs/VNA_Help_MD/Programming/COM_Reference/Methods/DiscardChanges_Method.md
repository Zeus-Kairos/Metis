## DiscardChanges Method

* * *

#### Description

|  Cancels changes that have been made to the Converter setup and reverts to
the previously-saved setup. Same as the Cancel button on the [mixer setup
dialog box](../../../Applications/MixerConverter_Setup.htm#Mixer_setup_help).  
---|---
