## DataCount Property

* * *

#### Description

|  Returns the total number of data points in the FIFO buffer.  
---|---  
  
####  VB Syntax

|  value = fifo.DataCount  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (Long Integer) Variable to store the returned number of data points.  
fifo |  A [FIFO](../Objects/FIFO_Object.md) Object  
  
#### Return Type

|  Long Integer  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = fifo.DataCount 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_DataCount(long *value)  
  
#### Interface

|  IFIFO
