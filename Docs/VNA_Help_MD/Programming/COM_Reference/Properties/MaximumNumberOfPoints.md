## MaximumNumberOfPoints Property

* * *

#### Description

|  Returns the maximum possible number of data points.  
---|---  
  
####  VB Syntax

|  value = obj.MaximumNumberOfPoints  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (Long) \- Variable to store the returned maximum value for number of points.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object) or A [GainCompression](../Objects/GainCompression_Object.md) (object)  
  
#### Return Type

|  Double  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.MaximumNumberOfPoints 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_MaximumNumberOfPoints(double * maximumNumberOfPoints );  
  
#### Interface

|  ICapabilities IGainCompression
