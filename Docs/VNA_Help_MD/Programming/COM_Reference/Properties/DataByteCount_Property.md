# DataByteCount Property

* * *

#### Description

|  Reads the FIFO data buffer byte count.  
---|---  
  
####  VB Syntax

|  value = fifo.DataByteCount  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (Long) Variable to store the returned data byte count.  
fifo |  A [FIFO](../Objects/FIFO_Object.md) Object  
  
#### Return Type

|  Long  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = fifo.DataByteCount 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_DataByteCount(long* bytecount);  
  
#### Interface

|  IFIFO2
