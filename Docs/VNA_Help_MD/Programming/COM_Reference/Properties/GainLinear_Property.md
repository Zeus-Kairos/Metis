## GainLinear Property

* * *

#### Description

|  Returns the Linear Gain result of a PSat marker search. Gain Linear =
Marker 1 - Y-axis value MINUS X-axis value  
---|---  
  
####  VB Syntax

|  gainLin = pSat.GainLinear  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
gainLin |  (double) - Variable to store returned value  
pSat |  A [PSaturation](../Objects/PSaturation_Object.md) (object)  
  
#### Return Type

|  Double  
  
#### Default

|  Not applicable
