## FilterCF Property

* * *

#### Description

|  Returns the Center Frequency result of the SearchBandwidth method.  
---|---

#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description

|  Double  
  
#### Default
