## Gain Property

* * *

#### Description

|  Returns the Gain result of the PNOP marker search. Pnop Gain = Pnop Out -
Pnop In.  
---|---  
  
####  VB Syntax

|  gain = pnop.Gain  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
gain |  (double) - Variable to store returned value  
pnop |  A [PNOP](../Objects/PNOP_Object.md) (object)  
  
#### Return Type

|  Double  
  
#### Default

|  Not applicable
