# InternalDCReceiverNames Property

* * *

#### Description

|  Returns a list of names of the internal DC receivers.  
---|---  
  
####  VB Syntax

|  value = cap.InternalDCReceiverNames  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  Variant containing one-dimensional array of string names.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.InternalDCReceiverNames 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_InternalDCReceiverNames(Variant *names);  
  
#### Interface

|  ICapabilities14
