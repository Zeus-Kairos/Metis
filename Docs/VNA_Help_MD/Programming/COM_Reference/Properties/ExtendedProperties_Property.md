## ExtendedProperties Property

* * *

#### Description

|  Provides access to the custom properties and methods of an external device.  
---|---  
  
####  VB Syntax

|  Set PSG = ExtDev.ExtendedProperties  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
PSG |  (Object) Variable to store the returned handle to an external device.  
ExtDev |  An [ExternalDevice](../Objects/ExternalDevice_Object.md) (object)  
  
#### Return Type

|  Object  
  
#### Default

|  Not Applicable
