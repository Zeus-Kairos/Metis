## SerialNumber Property

* * *

#### Description

|  Returns the serial number of the VNA.  
---|---  
  
####  VB Syntax

|  value = cap.SerialNumber  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (String) - Variable to store the returned value.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Return Type

|  String  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.SerialNumber 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_SerialNumber(BSTR * serial);  
  
#### Interface

|  ICapabilities11
