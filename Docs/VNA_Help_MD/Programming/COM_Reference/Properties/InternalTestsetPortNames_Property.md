## InternalTestsetPortNames Property

* * *

#### Description

|  Returns a list of internal test port names.  
---|---  
  
####  VB Syntax

|  value = cap.InternalTestsetPortNames  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  Variant containing one-dimensional array of string names.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.InternalTestsetPortNames 'Read
