## MinimumFrequency Property

* * *

#### Description

|  Returns the minimum frequency of the remote VNA.  
---|---  
  
####  VB Syntax

|  value = cap.MinimumFrequency  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (Double) - Variable to store the returned minimum frequency of the VNA.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Return Type

|  Double  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.MinimumFrequency 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_MinimumFrequency(double *pVal)  
  
#### Interface

|  ICapabilities
