## TestSetType Property

* * *

#### Description

|  Returns the Test Set Type (model) that was used for the Cal Set.  
---|---  
  
####  VB Syntax

|  TSType = calset.TestSetType  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
TSType |  (String) Variable to store the returned test set model.  
calset |  A [Cal Set](../Objects/CalSet_Object.md) object.  
  
#### Return Type

|  String  
  
#### Default

|  Depends on the test set.  
  
#### Example

|  TSType = calset.TestSetType
