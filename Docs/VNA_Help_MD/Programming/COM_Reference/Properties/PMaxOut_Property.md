## PMaxOut Property

* * *

#### Description

|  Returns the PMaxOut result of the PNOP and PSAT marker search. PMaxOut =
Marker 3 Y-axis value  
---|---  
  
####  VB Syntax

|  pMaxOut = pMark.PMaxOut  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
pMaxOut |  (double) - Variable to store returned value  
pMark |  A [PNOP](../Objects/PNOP_Object.md) (object) or [PSaturation](../Objects/PSaturation_Object.md) (object)  
  
#### Return Type

|  Double  
  
#### Default

|  Not applicable  
  
#### Examples

|  pMaxOut = pMark.PMaxOut 'Read [See example
program](../../COM_Example_Programs/Setup_PNOP_and_PSAT_Marker_Search.htm)  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_PMaxOut(double* pNewVal)  
  
#### Interface

|  IPNOP or IPSaturation
