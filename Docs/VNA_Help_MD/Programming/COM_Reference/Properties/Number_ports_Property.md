## Number Property

* * *

#### Description

|  Returns the VNA port number.  
---|---  
  
####  VB Syntax

|  value = port.Number  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
port |  A [Port](../Objects/Port_Object.md) (object)  
value |  (Long Integer) \- Variable to store the returned Port number.  
  
#### Return Type

|  Long  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = port.Number 'Read the value  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_Number([out,retval] long *port);
