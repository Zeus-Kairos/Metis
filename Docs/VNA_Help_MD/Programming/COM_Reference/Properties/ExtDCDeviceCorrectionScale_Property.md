# ExtDCDeviceCorrectionScale Property

* * *

#### Description

|  Returns the correction scaling for an external DC Source.  
---|---  
  
####  VB Syntax

|  value = extDC.ExtDCDeviceCorrectionScale  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
extDC |  An [ExternalDCDevice](../Objects/ExternalDCDevice_Object.md) (object)  
value |  (Double) Variable to store the returned correction scaling value.  
  
#### Return Type

|  Double  
  
#### Default

|  0  
  
#### Example

|  value = extDC.ExtDCDeviceCorrectionScale 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_ExtDCDeviceCorrectionScale(double *value)  
  
#### Interface

|  IExternalDCDevice3
