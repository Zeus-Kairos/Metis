## Values Property

* * *

#### Description

|  Returns an array of valid settings that can be used with the element
object. See a [list of configurable elements and
settings](../../RF_PathConfig.htm) for various VNA models.  
---|---  
  
####  VB Syntax

|  values = pathElement.Values  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
values |  (Variant array) Variable to store the array of valid settings for the element.  
pathElement |  A [PathElement](../Objects/PathElement_Object.md) (object)  
  
#### Return Type

|  Variant array  
  
#### Default

|  Not Applicable  
  
#### Examples

|  settings=pathElement.Values  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT Values( VARIANT* pValues );  
  
#### Interface

|  IPathElement
