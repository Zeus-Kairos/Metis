# PathElements Property

* * *

#### Description

|  Returns a string with the names of all valid RF path configuration
elements.  
---|---  
  
####  VB Syntax

|  elements = powerRange.PathElements  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
powerRange |  A [PowerRange](../Objects/Power_Range_Object.md) (object)  
elements |  (String) Variable to store RF path configuration element names.  
  
#### Return Type

|  String  
  
#### Default

|  Not Applicable  
  
#### Examples

|  elements = powerRange.PathElements  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_PathElements(BSTR *pathElements);
