# DimensionCount Property

* * *

#### Description

|  Read the highest dimension order in the multi-dimensional sweep.  
---|---  
  
####  VB Syntax

|  count = md.DimensionCount  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
count |  (long) Variable to store the returned dimension order value.  
_md_ |  A [MultiDimensionalSweep](../Objects/MultiDimensionalSweep_Object.md) (object) which belongs to a SA channel.  
  
#### Return Type

|  long  
  
#### Default

|  0  
  
#### Examples

|  count = md.DimensionCount 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_DimensionCount (long* count);  
  
#### Interface

|  IMultiDimensionalSweep
