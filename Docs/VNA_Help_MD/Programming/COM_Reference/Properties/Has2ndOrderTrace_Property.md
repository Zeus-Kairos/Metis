# Has2ndOrderTrace Property

* * *

#### Description

|  Returns the state of whether or not 2nd order parameters are measured. True
indicates 2nd order parameters are measured.  
---|---  
  
####  VB Syntax

|  value = imd.Has2ndOrderTrace  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
imd |  A [SweptIMD](../Objects/SweptIMD_Object.md) Object  
value |  Variable to store returned state.  
  
#### Return Type

|  Boolean  
  
#### Default

|  Not applicable  
|  value = imd.Has2ndOrderTrace 'Read
