## TotalIterations Property

* * *

#### Description

|  Returns the total number of iterations required by the last SMART sweep.
Returns number of power points for a 2D sweep.  
---|---  
  
####  VB Syntax

|  iter = gca.TotalIterations  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
iter |  (Integer) Variable to store the returned number of iterations.  
gca |  A [GainCompression](../Objects/GainCompression_Object.md) (object)  
  
#### Return Type

|  Integer  
  
#### Default

|  Not Applicable  
  
#### Examples

|  data = gca.TotalIterations
