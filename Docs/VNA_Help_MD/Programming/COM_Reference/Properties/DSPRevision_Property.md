## DspRevision Property

* * *

#### Description

|  Returns the DSP Revision number that is visible in the Help, About Network
Analyzer dialog box. [Learn more.](../../../S0_Start/HelpAbout.md)  
---|---  
  
####  VB Syntax

|  value = cap.DspRevision  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (String) - Variable to store the returned DSP revision number.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Return Type

|  String  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.DspRevision 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_DspRevision(BSTR *value);  
  
#### Interface

|  ICapabilities6
