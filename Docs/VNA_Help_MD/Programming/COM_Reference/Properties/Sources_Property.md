## Sources Property

* * *

#### Description

|  Returns the names of the configured DC sources for the specified channel.  
---|---  
  
####  VB Syntax

|  names = dc.Sources  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
names |  (Variant) Variable to store the returned DC source names.  
dc |  An [DCStimulus](../Objects/DCStimulus_Object.md) (object)  
  
#### Default

|  Not Applicable  
Example |  names = dc.Sources 'Read  
  
#### [C++](../../Learning_about_COM/c%2b%2b_and_the_com_interface.md) Syntax

|  HRESULT get_Sources(VARIANT * pValue);  
  
#### Interface

|  IDCStimulus
