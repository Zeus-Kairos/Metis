## DeviceInputPort Property

* * *

#### Description

|  Returns the VNA port number to be used for the DUT input.  
---|---  
  
####  VB Syntax

|  port = obj.DeviceInputPort  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
port |  (long) Variable to store the VNA port number of the DUT input.  
mixer |  A [Mixer](../Objects/IMixer_Interface.md) (object)  
  
#### Return Type

|  Long  
  
#### Default

|  1  
  
#### Examples

|  value = mixer.DeviceInputPort
