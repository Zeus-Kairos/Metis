## Name Property

* * *

#### Description

|  Returns the name of the current element object  
---|---  
  
####  VB Syntax

|  name = pathElement.Name  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
name |  (String) Variable to store the returned element name.  
pathElement |  A [PathElement](../Objects/PathElement_Object.md) (object)  
  
#### Return Type

|  String  
  
#### Default

|  Not Applicable  
  
#### Examples

|  name=pathElement.Name  
  
#### [C++](../../Learning_about_COM/c++_and_the_com_interface.md) Syntax

|  HRESULT get_Name(BSTR* ppName)  
  
#### Interface

|  IPathElement
