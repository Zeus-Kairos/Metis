# InternalDCSourceCount Property

* * *

#### Description

|  Returns the number of internal DC sources in the analyzer.  
---|---  
  
####  VB Syntax

|  value = cap.InternalDCSourceCount  
  
#### Variable

|

#### [(Type)](../../Learning_about_COM/COM_Data_Types.md) \- Description  
  
value |  (Long) - Variable to store the returned value.  
cap |  A [Capabilities](../Objects/Capabilities_Object.md) (object)  
  
#### Return Type

|  Long  
  
#### Default

|  Not Applicable  
  
#### Examples

|  value = cap.InternalDCSourceCount 'Read
