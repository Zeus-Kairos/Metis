Accessories are available in these connector types:

  * 50 ohm Type-N

  * 75 ohm Type-N

  * 3.5 mm

  * 7 mm (APC-7)

  * 7-16

  * 2.92 mm

  * 2.4 mm

  * 1.85 mm

  * 1 mm

Test port cables and a calibration kit are necessary for a complete
measurement system.

A verification kit is used to verify corrected system performance.

[See the connector type for each VNA model](Configurations.md)
